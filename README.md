## 🚀 Run Locally
```bash
git clone https://github.com/Erharshita-cloud/my-static-website.git
cd my-static-website
# Open index.html in your browser
```

---

## 🛠️ How to Deploy on GitHub Pages

1. Push your code to a GitHub repo
2. Go to **Settings → Pages**
3. Set source branch to `main`
4. GitHub generates a live URL instantly ✅
